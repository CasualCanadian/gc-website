VisualCS

VisualCS is a program that I created for a software package for Porewater Solutions. It also doubled as my grade 12 computer science project. It reads comma delimited files, and plots the visuals according to the input files. The plot is a cross-section of a site, monitoring the levels of pollutants in the groundwater. Now again I'm no civil engineer so I don't fully understand all of this jargon, but the program still does what it needs to do. Although I completed the project for my computer science class, there are still more features to be added and some polishing required before the software becomes available. I plan on continuing work on this at some point in the near future.
